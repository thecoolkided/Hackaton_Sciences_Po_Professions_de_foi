### Définition de batch

Le batch permet de diviser une quantité importante de données en plusieurs lots. 
Ces petits lots sont ensuite traités un par un, ce qui rend le processus plus fluide et progressif. 

Deux dossiers contiennent les batchs effectués sur les résultats des deux OCR Mistral et fitz. Un bacth contient 50 professions de fois différentes. 

**Exemple**
batch_test_mistral_O = 50 professions de fois.
