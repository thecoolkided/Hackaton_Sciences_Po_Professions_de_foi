Résultats de Mistral Large 3 sur le batch_0 de l'OCR Fitz. 

**Explications**

Ce dossier contient deux fichiers : results_test_fitz_raw.csv et results_test_fitz_clean.csv. 
Le fichier results_test_fitz_raw.csv contient les résultats non-corrigés obtenus après le lancement de Mistral Large 3 sur le batch_0 de l'OCR de fitz. Nous avons fait en sorte d'afficher erreurs sont affichées en gras.
Le fichier results_test_fitz_clean.csv contient les mêmes résultats sans les erreurs. La correction est également affichée en gras.
